# 💖 Smart Maternal Safety & Empowerment Network

A comprehensive, production-ready maternal healthcare ecosystem built with **Flask** and **SQLite**. Designed for rural accessibility and premium healthcare management.

## 🚀 Key Features
- **Maternal Tracking**: Longitudinal tracking for Pregnancy, Baby Care, and Recovery.
- **AI Health Suite**: AI Assistant chat and Risk Prediction for hypertension, anemia, and diabetes.
- **Emergency SOS**: One-tap geolocation-based emergency alert system.
- **Smart Scheduler**: Integrated reminders for medication and nutrition.
- **Medical Network**: Connect with certified doctors and order prenatal supplements.
- **Empowerment**: Direct directory for Government Schemes and a Maternal Job Marketplace.

## 🛠️ Tech Stack
- **Backend**: Python (Flask), SQLAlchemy (SQLite)
- **Frontend**: Vanilla HTML/CSS/JS (Modern Aesthetic, Gradient Theme)
- **Security**: Flask-Login session management, Password Hashing.

## 🏥 Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Access at: `http://localhost:5000`
